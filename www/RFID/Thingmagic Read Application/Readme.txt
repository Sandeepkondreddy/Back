--Application for Tag Reading application for Thingmagic devices(Astra-EX,USB Reader)--
1. Application need to Read the Tags for the Reader.
2. Application has to validate the tags by comparing the tag details with Byroad and Internal Trucks.
3. Application has to capture the details save the details if truck is related to by road Fleet--  dbo.MTFX9500BYROAD
4. Application has to capture the details save the details if truck is related to Internal Fleet--  dbo.MTFX9500InternalFleet
5. Application has to capture the details save the details if truck is not related to --  dbo.MTFX9500JunkData
